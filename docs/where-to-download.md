# Where to download the ISO artifact

1. Open GitHub and navigate to the repository page.
2. Click **Actions** and choose the **build-iso** workflow.
3. Open the latest run (or the run you triggered) and wait for it to finish.
4. Scroll to the **Artifacts** section at the bottom of the run summary.
5. Download the **cachyos-iso** artifact; it contains the generated `.iso` file.
